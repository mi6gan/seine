This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

### Установка:
1. Node.js
```bash
apt-get install nodejs
```
1. Менеждер пакетов
```bash
npm i -g yarn
```
1. Зависимости
```bash
yarn install
```

### Команды:
1. Запуск storybooks на localhost:9090
```bash
yarn start
```
1. Запуск тестов
```bash
yarn test
```
1. Сборка статики в `./storybook-static/`
```bash
yarn build
```
